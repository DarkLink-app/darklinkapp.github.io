Approach

Built for those who don’t trust.

DarkLink starts from a simple premise: a compromised server should reveal nothing. Encryption is total, the network is Tor, and the infrastructure is yours.

End-to-end encryption

Messages are encrypted on your device before being sent. The server never sees their content — only data it cannot read.

Tor-only network

Every DarkLink server is a .onion address. No visible IP, no open ports, no DNS. The Tor network is the only path in.

Your server, your rules

You install and control your own instance. Who can register, how data is handled, who has access — everything stays in your hands.

Renewable identities

Every user can renew their identity at any time without losing their conversations or contacts.

Cross-server messaging

Users on different servers can communicate directly, server to server, without going through any third party.

Zero public trace

No open profile, no public feed, no exposed metadata. DarkLink does not exist on the visible Internet.

Open source

DarkLink is fully open source. The code, documentation and install script are available on GitHub.
